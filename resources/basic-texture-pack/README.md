# Basic Texture Pack

This is a starter structure for a Minecraft Java resource pack.

## Common places to add files

- `assets/minecraft/textures/block/` for block textures
- `assets/minecraft/textures/item/` for item textures
- `assets/minecraft/textures/entity/` for mob and entity textures
- `assets/minecraft/models/block/` for custom block model JSON files
- `assets/minecraft/models/item/` for custom item model JSON files
- `assets/minecraft/lang/en_us.json` for text overrides

## Notes

- Add an optional `pack.png` file in this folder to give the pack an icon.
- Replace the placeholder files with your own textures or JSON model files.
